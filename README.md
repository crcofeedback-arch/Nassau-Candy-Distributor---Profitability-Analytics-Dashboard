# ☕ Afficionado Coffee Roasters - Product Optimization & Revenue Analysis

## 📊 Project Overview
Interactive Streamlit dashboard analyzing product revenue contribution, category performance, and menu optimization for Afficionado Coffee Roasters. Identifies hero products, underperformers, and provides data-driven recommendations.

## 🎯 Problem Statement
The organization lacks visibility into product popularity vs revenue, category-level revenue dependency, and low-impact menu items. Menu decisions are intuition-driven, leading to inefficiencies.

## ✨ Features
- Product ranking by revenue & sales volume
- Category performance (Coffee, Tea, Chocolate)
- Pareto 80/20 revenue concentration analysis
- Product efficiency scoring with quadrant classification
- Menu simplification recommendations

## 🛠️ Tech Stack
- Python 3.14
- Streamlit
- Pandas
- Plotly
- NumPy

## 📁 Dataset
- 12,847 transactions
- 35 unique products
- 5 store locations
- Categories: Coffee, Tea, Chocolate
